🤖 AI Talent Agent

🚀 Project Overview

The AI Talent Agent is a smart recruitment assistant designed to automate the process of candidate screening and matching. It takes a Job Description (JD) as input and intelligently analyzes candidate profiles to identify the most suitable candidates.

This project helps recruiters save time by reducing manual effort and improving the accuracy of candidate selection using basic AI-driven logic.

---

💡 Key Features

- 📄 Input Job Description (JD)
- 📊 Analyze candidate data from CSV file
- 🤖 Match candidates based on skills and requirements
- 📈 Display ranked results of best-fit candidates
- 🌐 Simple web interface using Flask

---

🛠️ Tech Stack

- Python
- Flask (Web Framework)
- Pandas (Data Processing)
- HTML & CSS (Frontend)
